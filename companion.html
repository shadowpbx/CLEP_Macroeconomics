edit_note How to Edit This Companion

To populate this study companion with your actual course materials, follow these simple steps:

  1. Open the file companion.html in your favorite code editor (e.g., VS Code).
  2. Modify the title, headings, and text inside the card sections to write your tutorials.
  3. Drop diagrams and graphics into your local workspace and link them using standard HTML <img> tags.
  4. Save the file and commit/push it to your GitHub repository alongside the player.
